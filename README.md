# Japonia - prosta strona WWW

To jest prosty projekt strony o Japonii przygotowany w HTML, CSS i JavaScript.
Strona zawiera informacje o kraju, kulturze, popularnych miejscach oraz element dynamiczny.

## Pliki

- `index.html` - struktura strony i tresc.
- `style.css` - wyglad strony, kolory, uklad i responsywnosc.
- `script.js` - prosty element dynamiczny: data oraz losowanie ciekawostki.

## Element dynamiczny

Strona jest dynamiczna, poniewaz JavaScript zmienia tresc bez odswiezania strony.
Po kliknieciu przycisku "Losuj ciekawostke" pojawia sie inna ciekawostka o Japonii.
Dodatkowo data na stronie jest pobierana automatycznie z przegladarki.

## Jak uruchomic

Najprosciej otworzyc plik `index.html` w przegladarce.
Projekt nie wymaga instalowania dodatkowych bibliotek.

## Co mozna powiedziec na obronie

- HTML odpowiada za tresc i podzial strony na sekcje.
- CSS odpowiada za wyglad, kolory, zdjecia w tle i dopasowanie do telefonu.
- JavaScript dodaje dynamicznosc: wpisuje aktualna date i losuje ciekawostke.
- Meta tag `author` podaje autora projektu.
- Meta tag `description` opisuje strone.
- Meta tag `viewport` sprawia, ze strona dobrze skaluje sie na telefonach.
