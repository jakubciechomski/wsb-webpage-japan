# Japonia - projekt strony WWW

Prosta strona o Japonii.

## Uzyte technologie

- HTML
- CSS
- JavaScript

## Co znajduje sie na stronie

- podstawowe informacje o Japonii,
- tabela z najwazniejszymi danymi,
- sekcja o kulturze,
- znane miejsca,
- prosty formularz z powitaniem po japonsku.

## Element dynamiczny

Strona ma prosty element dynamiczny w JavaScript.
Po wpisaniu imienia i kliknieciu przycisku pojawia sie powitanie zapisane alfabetem lacinskim.
